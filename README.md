# Mel Tejo

Mel Tejo is a SwiftUI iOS app concept built around tejo, the national sport of Colombia.

It includes:
- `Train`: a 50-second reaction mode for learning mecha, mano, bocin, and moĂąona
- `Match`: a live score tracker for real tejo sessions
- `Discover`: official links, history, and starter pathways into the sport
- `Insights`: saved matches, shot breakdowns, and training accuracy

Recommended App Store categories:
- Primary: `Sports`
- Secondary: `Games`
